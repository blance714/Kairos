# Kairos 时间管理应用 - 需求文档

## 项目概述

Kairos 是一款个人定制的 iOS 时间管理应用，目标是通过弹性规则控制成瘾性应用的使用，而不是简单粗暴的硬性限制。

核心设计原则：**规则要让人愿意遵守，而不是想要绕过或卸载。**

## 管控应用

| 应用 | 分类 |
|------|------|
| 小红书 | 通用组 |
| 微博 | 通用组 |
| Bilibili | 通用组 |
| 小说软件（具体 App 待定）| 独立 |

## 模式定义

### 1. 早晨模式 🌅

**触发条件**：
- 当前时间 ≥ 08:00
- 且 睡眠专注模式解除后不足 1 小时

**规则**：
- 管控应用**完全锁定**，不可使用

**目的**：防止早上赖床刷手机，导致工作日迟到或周末被毁掉

**退出条件**：睡眠专注模式解除满 1 小时 → 进入普通模式

---

### 2. 普通模式 ☀️

**触发条件**：
- 不满足早晨模式条件
- 且 不满足晚上模式条件

**规则**：
- 连续使用 15 分钟后 → 进入冷却
- 冷却 30 分钟后 → 可再次使用
- 循环

**说明**：这是白天/在外的默认模式，允许间歇性使用但防止沉迷

---

### 3. 晚上模式 🌙

**触发条件**：
- 当前位置在家（GPS 定位，出租屋范围）
- 且 当前时间 ≥ 22:00

**初始规则（未激活）**：
- 与普通模式相同：15 分钟使用 + 30 分钟冷却循环

**激活条件**：
- 连续 30 分钟未使用任何管控应用

**激活后规则（额度制）**：

| 分类 | 额度 |
|------|------|
| 小说软件 | 独立 45 分钟 |
| 通用组（小红书+微博+B站）| 共享 20 分钟 |

**设计意图**：
- 洗澡自然会触发"30分钟不用"，从而激活额度
- 不强制洗澡，但用冷却期的无聊推动用户去洗
- 小说需要沉浸感所以独立额度，其他碎片化应用共享额度

---

## 模式优先级

```
早晨模式 > 晚上模式 > 普通模式
```

即：如果同时满足多个条件，按此优先级判断。

## 状态流转图

```
[睡眠专注模式中]
        │
        ▼ 解除
[等待 1 小时] ──08:00前解除──▶ [普通模式]
        │
        │ 08:00后解除
        ▼
[早晨模式：锁定] ──满1小时──▶ [普通模式]
        
[普通模式] ──在家+22:00后──▶ [晚上模式：未激活]
                                    │
                                    │ 30分钟未使用
                                    ▼
                              [晚上模式：已激活]
                                    │
                                    │ 额度用完 / 次日
                                    ▼
                                 [重置]
```

## 需要调研的技术问题

### iOS 能力边界

1. **Screen Time API**
   - 能否监听单个 App 的实时使用时长？
   - 能否在使用 15 分钟后触发锁定？
   - 能否实现"冷却 30 分钟后自动解锁"？

2. **专注模式 API**
   - 能否读取"睡眠专注模式"的开启/关闭状态？
   - 能否获取解除的时间点？

3. **GPS / Geofencing**
   - 能否设置家的位置范围（geofence）？
   - 进入/离开区域时能否触发回调？
   - 与 Screen Time 的联动是否可行？

4. **后台运行**
   - 计时器在后台能否正常工作？
   - "连续 30 分钟未使用"如何准确检测？

5. **权限与限制**
   - 是否需要 MDM 配置描述文件？
   - 普通开发者账号能否实现，还是需要企业签名？
   - 有哪些功能只能通过快捷指令实现？

## 暂不实现的功能

以下功能在讨论中被否决，记录原因以备后续参考：

| 功能 | 否决原因 |
|------|----------|
| 一次性延长 30 分钟（消耗晚上额度）| 短视使用后晚上没得用 → 烦躁卸载 App |
| 查东西通行证 | 很容易变成绕过冷却的万能借口 |
| 晚间 Checklist | 便利贴更实用，且手机充电时看不了手机 |
